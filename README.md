# amsystems

Este aplicativo é voltado para profissionais de Odontologia e Clínicas Odontológicas. O aplicativo apresenta uma interface que não necessita de instalação na máquina, sendo utilizada somente em banco de dados. A gestão da clínica ou do profissional é feita através de agendamento, via WhatsApp, com o ontrole 
financeiro completo, orçamentos e tratamentos a realizar e serem realizados, além de odontograma completo para cadastramento dos tratamentos.
